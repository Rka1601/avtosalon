 
# Автосалон - Django проект

## Описание проекта
Веб-приложение для автосалона с каталогом автомобилей, заявками на покупку и осмотр.

## Технологии
- Django 5.2.8
- SQLite (разработка)
- Bootstrap
- Pillow для обработки изображений
